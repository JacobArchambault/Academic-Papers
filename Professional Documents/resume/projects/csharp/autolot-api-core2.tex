\datedsubsection{2020-present}{Auto lot MVC store}{}{A containerized three-project .NET 5 MVC \href{https://github.com/jacobarchambault/autolotmvc_core2}{solution} consisting of a website front-end, a data access layer, and a model layer. Makes use of .NET 5 records, razor view components, custom tag helpers, exception filters, custom routing, and LINQ queries over objects linked via entity framework core to a SQL Server relational database. Expanded from a \emph{Pro C\# 7} tutorial.}
