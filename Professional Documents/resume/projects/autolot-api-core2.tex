\datedsubsection{2020-present}{Auto lot MVC store}{}{A four-project .NET Core MVC \href{https://github.com/jacobarchambault/autolotapi_core2}{solution} consisting of a website front-end, an API project, a data access layer, and a model layer. Makes use of razor view components, custom tag helpers, exception filters, custom routing, and LINQ queries over objects linked via entity framework core to a SQL Server relational database. Expanded from a \emph{Pro C\# 7} tutorial.}
